141 环形链表

给定一个链表，判断链表中是否有环。

为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。

示例 1：

输入：head = [3,2,0,-4], pos = 1
输出：true
解释：链表中有一个环，其尾部连接到第二个节点。

解法

  • 快慢指针，若没有环，那么肯定fast最终指向None；若有环，类似于操场场景，跑快的同学最终会和跑慢的同学相遇。此时他们跑的距离差一圈了。注意的是，比较的是结点Node而不是结点的值val。
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def hasCycle(self, head: ListNode) -> bool:
        if head == None: return False
        slow,fast = head,head

        while slow and fast:
            slow = slow.next
            if fast.next:
                fast = fast.next.next
            else:
                return False
            if slow==fast:
                return True
        return False
